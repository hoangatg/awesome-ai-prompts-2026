# Contributing to Awesome AI Prompts 2025

Thank you for contributing! Every quality prompt helps the AI community.

## How to Contribute

### Adding a New Prompt

1. **Fork** this repository
2. Find the appropriate category in `prompts/[category]/README.md`
3. Add your prompt using the template below
4. Submit a **Pull Request**

### Prompt Template

```markdown
## [Prompt Name]

**Best For:** [Primary use case]
**Works With:** ChatGPT, Claude, Gemini

> [The full prompt text here]

**Pro Tips:**
- [Tip for getting better results]
- [How to customize for specific needs]
```

### Quality Checklist

- [ ] Prompt has been tested with at least 2 different AI models
- [ ] Uses clear, specific language (not vague instructions)
- [ ] Includes structured output format when applicable
- [ ] Has practical "Pro Tips" for customization
- [ ] No harmful, misleading, or unethical prompts
- [ ] Describes a real use case, not novelty
